# Pydantic Models & Validation Playground

This repository demonstrates how Pydantic enforces data contracts and validation in Python.

## Concepts Covered
- Required vs optional fields
- Type enforcement
- Field constraints
- Nested models
- Custom validators
- Request vs response schemas
- Automatic validation errors
