# Course Explorer API (FastAPI Project)

A beginner-friendly FastAPI project created to clearly demonstrate the use of **Path Parameters** and **Query Parameters** within a real API scenario.

---

##  Project Overview

The **Course Explorer API** allows users to:

- Fetch a course by its unique ID using **Path Parameters**
- Filter courses by category, level, rating, or limit using **Query Parameters**
- Combine both Path + Query parameters to list courses by instructor

This project is perfect for understanding how FastAPI handles different types of parameters and how to structure a small API.

---

## Tech Stack

- **Python 3.10+**
- **FastAPI**
- **Uvicorn**
- **Pydantic**
- **Virtual Environment (venv)**
